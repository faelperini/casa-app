## Sobre o projeto

**Casa** é uma plataforma colaborativa para moradores de uma mesma residência. Cada grupo representa uma casa, e seus membros compartilham um espaço comum para organizar o dia a dia — sem planilhas, sem grupos de WhatsApp confusos.

O acesso é feito exclusivamente via conta Google, garantindo uma experiência simples e segura. Grupos podem ser públicos (acesso por código de convite) ou privados (protegidos por senha).

---

## Funcionalidades

- **Autenticação** via Google OAuth — sem senhas para gerenciar
- **Grupos domésticos** com código de convite único
- **Grupos privados** com proteção por senha
- **Lista de compras** colaborativa com itens marcáveis em tempo real
- **Controle de débitos** com divisão automática de valores entre moradores
- **Livro de receitas** compartilhado com ingredientes, modo de preparo e foto
- **Perfil de usuário** com foto, bio e data de nascimento
- **Upload de imagens** para grupos, receitas e perfis
- **Papéis de membro** — administrador e membro — com permissões distintas
